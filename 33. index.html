▶ 교과별 등급 요약
—
▶1 내 등급 입력
학기를 펼쳐 과목별 등급을 확인/수정하세요. 체크 후 '선택 삭제'로 제거, '+ 과목 추가'로 추가 가능. 입력은 자동 저장됩니다.
※ 1~9등급 숫자 입력. 진로선택/성취도 과목은 A, B, C 문자 입력 가능 (A→1, B→5, C→8 자동변환)
※ 교과 분류는 대학별 산출에 사용됩니다. 빈 행은 무시됩니다.
▶2 대학·전형 목록
필요 없는 전형은 체크 후 '선택 삭제'로 제거하세요. 삭제 상태는 자동 저장됩니다.
